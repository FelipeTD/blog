# blog
Projeto Spring utilizando Kotlin

## Conceitos abordados nesse projeto
  - Criação de um novo projeto utilizando o Initializr com as dependências Spring Web, Mustache, Spring Data JPA, H2 Database, Spring Boot DevTools e Gradle
  - Criação de um Controller com testes unitários utilizando JUnit 5
  - Criação de arquivos de extensão para classes utilitárias
  - Persistência de dados utilizando JPA com testes unitários utilizando JUnit 5
  - Criação de uma HTTP API para acessar os dados via API com testes unitários utilizando Mockk por funcionar melhor com o Kotlin
  - Criação de um arquivo de propriedades para guardar os dados da aplicação como somente leitura
